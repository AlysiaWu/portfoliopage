Doctor's Appointments

• This is a website for booking doctor's appointments, allows users to enter complaint and time and date wish to be seen by the doc. 1-person project utilizing MVC project methodology to create a fully functional website from scratch. Using AngularJS for front end and Express and Node js for server, and MongoDB for data store.
• Contribution: Setup the backbone of framework with basic scripts and routings. Implement angular at the front end, express and node at the server, and the mongoose query at the back end.
• Technologies: MongoDB, Express, Angular, Node, Mongoose, HTML5, CSS3, JavaScript, Git/Github
